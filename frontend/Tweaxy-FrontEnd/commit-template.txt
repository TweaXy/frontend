🐞 BUG -> Bug Fix
➕ FR -> adding a feature 
🎨 REF -> refactor
📝 DOC -> Documentaion/README
⏪️ RVT -> Reverting changes
📦️ PKG -> Package/dependecies changes (upgrade/downgrade)
⚡️ PERF -> performance improvement.

example:

[ ➕ FR ] Adding login Page; adding startup page;
[ 📦️ PKG ] Adding login Page; adding startup page;
